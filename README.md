## Requisitos
 - ***Docker*** 
 - ***docker-compose***

## Execução
Para executar não é necessário configuração, basta subir primeiramente o Hadoop com o comando:

```shell
$ docker-compose up --build hadoop
```

E em seguida, subir a aplicação:

```shell
$ docker-compose up --build
```
